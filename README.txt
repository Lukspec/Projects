Arquivos com extensão form, são arquivos xml usados para programar a interface da tela.
E os arquivos Java, são a programação principal da tela.

Programa feito no Apache NetBeans, programado em java. 
Com salvamento de arquivos local sem o uso de banco de dados.

Projeto da faculdade com o intuito de simular um programa de uma farmacia real.
Com registro de funcionarios, estoque e clientes.
Realizando vendas de produtos selecionados.
